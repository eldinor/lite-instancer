@litools/instancer examples

Run one dev server with npm run dev, then open any demo from here.

  • Basic Thin Instances Stable IDs, transforms, colors, removal, and new instances.
  • Primitive Box Field Boxes, colors, batch transforms, selection.
  • Primitive Sphere Cloud Metadata, group visibility, motion.
  • Primitive Mixed Playground Multiple primitive sets with one shared picking flow.
  • Visibility Layers Active-count and scale-zero visibility strategies.
  • Raw Batch Streaming Batch writer vs controlled raw buffer edits.
  • BoomBox Grid GLB hierarchy instance grid, picking, add/remove.
  • BoomBox Picker GLB picking and stable ID deletion after slot swaps.
  • BoomBox Rebuild Growth Hierarchy pool growth by rebuild while IDs survive.
  • Shark School Shared Animation Animated GLB school behavior with one shared animation phase.
  • Shark Phase Buckets VAT animation variation with per-instance phase and fps offsets.
  • Shark Clip Mixer Per-instance VAT clip assignment with selected shark and lane controls.
  • Dist VAT Acrobatic Plane Imports the built package from dist and animates acrobatic plane VAT instances.
  • Ready Player Basic Animation Loads and loops the supplied animated Ready Player GLB.
  • Ready Player VAT Sword Sync VAT avatar parts with a thin sword instance synchronized to the animated right hand.
  • Samba Girl VAT Sword Sync HVGirl Samba VAT instances with thin swords synchronized across the glTF RH to Lite LH conversion.
  • GLB VAT Socket Configurator Choose a VAT rig socket, tune a full GLB attachment transform, and export JSON plus TypeScript.
  • Unarmed VAT Arena Crowd 3,000-capacity local-GLB VAT fighters in three independently baked crowd groups using 9 selected clips.
  • Thin Instance Outline Gallery Primitives, imported GLB/glTF assets, live skeletal outlines on an animated fan, selection, effects, and the standalone API.
