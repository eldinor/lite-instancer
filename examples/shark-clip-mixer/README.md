# Shark Clip Mixer

Uses:

```txt
https://assets.babylonjs.com/meshes/shark.glb
```

Demonstrates per-instance VAT clip assignment on top of phase/fps variation.

The example assigns clips by lane when more than one animation group is available. Use `cycle selected clip` to move one shark to the next clip, or `cycle selected lane` to move every shark in the selected lane.

Run from the root examples index with:

```sh
npm run dev
```

Open `/examples/shark-clip-mixer/`.
