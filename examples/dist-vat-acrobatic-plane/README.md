# Dist VAT Acrobatic Plane

Asset:

https://assets.babylonjs.com/meshes/Demos/optimized/acrobaticPlane_variants.glb

This example imports `@litools/instancer` APIs from the local built package output in `dist`, not from `src`.

It demonstrates VAT animation with the acrobatic plane GLB, per-instance clip/phase/fps variation, screen-space logical picking, visibility controls, and batched transform updates.

Run from the root examples index with:

```sh
npm run dev
```

Open `/examples/dist-vat-acrobatic-plane/`.

Run `npm run build` before using this example if `dist` is stale.
