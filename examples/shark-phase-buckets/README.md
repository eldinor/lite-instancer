# Shark Phase Buckets

Uses:

```txt
https://assets.babylonjs.com/meshes/shark.glb
```

Demonstrates practical animation variation with one VAT-backed shark mesh and per-instance phase/fps offsets.

Use `shuffle phases` to reassign offsets across the active animation clip. Use `next animation` to switch the active animation group and rebucket the school.

Run from the root examples index with:

```sh
npm run dev
```

Open `/examples/shark-phase-buckets/`.
